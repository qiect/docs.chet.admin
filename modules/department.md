# 部门管理

## 功能特性

- 部门 CRUD（树形结构）
- 负责人、联系电话、邮箱
- 排序、启用/禁用
- 作为数据权限过滤的基础

## 后端接口

| 方法 | 路径 | 说明 |
| ---- | ---- | ---- |
| GET | `/departments` | 部门树 |
| POST | `/departments` | 创建 |
| PUT | `/departments/{id}` | 更新 |
| DELETE | `/departments/{id}` | 删除 |

## 实现细节

- 部门采用树形结构（`ParentId` 自关联，0 表示顶级）
- 部门是数据权限过滤的基础，角色的 `DataScope` 通过部门进行数据范围控制
- 删除部门时需检查是否存在子部门与关联用户
